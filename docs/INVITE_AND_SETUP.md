# Cómo invitar HackLabH Bot

## Enlace oficial

- https://panel.hacklabh.xyz/invite

## Permisos recomendados

- Administrador (para configuración inicial rápida), o
- Permisos granulares según tu política del servidor.

## Requisitos de aplicación

- Scope de invitación: `bot applications.commands`
- Para panel web: OAuth con `identify guilds`

## Verificación rápida

1. Invita el bot con el enlace oficial.
2. Ejecuta `/help` o `!help`.
3. Abre el panel: https://panel.hacklabh.xyz
4. Inicia sesión con Discord y selecciona tu servidor.
